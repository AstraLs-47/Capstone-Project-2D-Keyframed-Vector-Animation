# Ethiopian Warrior Vector Animation

## Overview

This project is a real-time 2D vector animation developed as part of a computer graphics capstone project. The animation features a stylized Ethiopian warrior created entirely from geometric primitives and rendered dynamically without the use of pre-rendered video or sprite sheets.

The primary objective of the project is to demonstrate the application of computer graphics principles by constructing an articulated character and animating it through mathematical transformations and interpolation techniques. The project also incorporates visual elements inspired by Ethiopian culture through color choices, patterns, and character design.

## Features

- Real-time rendering using OpenGL
- Composite vector-based character built from geometric primitives
- Hierarchical transformations for articulated body movement
- Smooth keyframe-based animation
- Timer-driven animation updates
- Programmatically generated cultural patterns and colors
- Modular and organized code structure

## Technologies Used

- **Programming Language:** Python
- **Graphics Library:** OpenGL (PyOpenGL)
- **Window Management:** GLUT
- **Development Environment:** Python 3.x
